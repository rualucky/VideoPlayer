Javascript API

Do những khác biệt giữa HTML5 và Flash, MeCloud Player cung cấp API giúp cho việc tương tác với video dựa theo hai công nghệ trên trở nên dễ dàng hơn.

Để dễ dàng sử dụng API, nên tạo MeCloud Player như sau:

<div id="player"></div>
<script>
    var playerInstance = meplayer('player');
    playerInstance.setup({
        primary: "html5",
        title: "MeCloud Video Demo",
        image: "http://mecloud.vn/images/android.png",
        width: 640,
        aspectration: "16:9",
        file: "http://mecloud.vn/videos/android.mp4",
    });
    playerInstance.play();
</script>

Một số API được hỗ trợ:

  • on(eventName, function)

    Cài đặt một sự kiện.

    
<script>
    playerInstance.on("hello", function(name){ // tạo 1 sự kiên tên là "hello", biến truyền vào là "name"
        console.log('Xin Chào ' + name);
    });
</script>
  • trigger(eventName, data)

    Thực thi một sự kiện.

    
<script>
    playerInstance.trigger("hello", "Bạn"); 
    // tìm và thực thi sự kiện tên "hello", truyền giá trị "Bạn" vào "name". 
    // Trên console sẽ in ra câu Xin Chào Bạn
</script>
  • play()

    phát video.

  • pause()

    tạm dừng video.

  • getVolume()

    trả về âm thanh hiện tại của video.

  • setVolume(value)

    cài đặt âm thanh cho video.

  • setMute()

    tắt âm thanh của video.

  • getDuration()

    trả về thời lượng của một video(tính bằng giây).

  • getCurrentTime()

    trả về thời điểm hiện tại khi video đang phát.

  • resize(chiều rộng, chiều cao)

    thay đổi kích thước của video.

  • ...

Chúng tôi luôn sẵn sàng phát triển thêm nhiều API khác nhau theo yêu cầu của các bạn.

Ví dụ về code cài đặt và API

<div id="player"></div>
<script>
    var playerInstance = meplayer('player');
    playerInstance.setup({
        primary: "html5",
        title: "MeCloud Video Demo",
        image: "http://mecloud.vn/images/android.png",
        width: 640,
        aspectration: "16:9",
        file: "http://mecloud.vn/videos/android.mp4",
    });
</script>
<button onclick="playerInstance.play()">Play</button>
<button onclick="playerInstance.pause()">Pause</button>
<button onclick="playerInstance.setMute()">Tắt âm thanh</button>
<button onclick="playerInstance.setVolume(80)">Mở âm thanh</button>
<button onclick="playerInstance.resize(560,320)">Thay kích thước video</button>
