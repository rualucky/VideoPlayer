Cách nhúng MeCloud Player vào website của bạn

Phần viết này sẽ hướng dẫn các bạn cách nhúng video vào website vô cùng đơn giản. Ngoài ra, MeCloud Player còn hỗ trợ nhiều chức năng khác như chèn quảng cáo vào video, hỗ trợ streaming dạng RTMP, DASH, HLS.... Để biết thêm thông tin, vui lòng lựa chọn chủ để tương ứng.

Ví dụ về MeCloud Player

Video trên sử dụng file MP4, vì đây là dạng format hỗ trợ từ desktop đến các thiết bị di động(mobile). Chúng tôi khuyên các bạn tốt nhất là luôn dùng video MP4 thay vì FLV(do các file FLV không được hổ trợ trên các thiết bị di động).

Để nhúng một MeCloud Player, các bạn làm theo các bước sau:

Nhúng MeCloud Player Library (Javascript) vào website của bạn.

<script type="text/javascript" src="http://mecloud.vn/MeCloud.js"></script>

Tạo code để nhúng video, bạn có thể nhúng đoạn code HTML này vào vị trí mà bạn muốn video xuất hiện trong website của bạn.

<div id="player"></div>
<script>
    meplayer('player').setup({
        primary: "html5",
        title: "MeCloud Video Demo",
        image: "http://mecloud.vn/images/android.png",
        width: 640,
        height:300,
        aspectration: "16:9",
        file: "http://mecloud.vn/videos/android.mp4",
    });
</script>

Trong đoạn code trên, thẻ <div> được dùng như một ngăn chứa MeCloud Player với hàm setup() dựa vào ID của <div>. Với một số lựa chọn cơ bản như:

  • primary - lựa chọn phát video hỗ trợ Flash hoặc HTML5.
  • title - tên tựa đề của video.
  • image - ảnh hiển thị của video.
  • width - chiểu rộng của video.
  • height - chiều cao của video.
  • aspectration - tỉ lệ chiểu rộng / chiều cao của video. Khi sử dụng aspectration thì lựa chọn height sẽ bị bỏ qua.
  • file - video sẽ được phát.

Còn rất nhiều lựa chọn cấu hình khác nằm trong tùy chỉnh video.
